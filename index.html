PoE1 • Witch Necromancer

League Start Mirage — SRS (v2 interativa)

Versão com navegação por abas para consulta rápida durante campanha, mapas e transição endgame.

Resumo

Build sólida para começar liga: progressão confortável, bom custo-benefício no early game e escalonamento forte no endgame com poison/chaos.

Classe: Witch Ascendancy: Necromancer Main Skill: Summon Raging Spirit Meta: Mirage League Start

Ato 1–2

  • Início: Rolling Magma, migrando para SRS cedo.
  • Suporte de leveling: Holy Flame Totem + Combustion.
  • Mobilidade: Frostblink (ou Flame Dash).
  • Matar todos os bandidos no Ato 2.

Ato 3–4

  • Ativar Purity of Elements e Determination quando possível.
  • Ajustar atributos (DEX/STR) no gear para requisitos.
  • Pegar Unleash e preparar links para transição.
  • Entrar no Lab 1 no timing seguro.

Core (early)

  • SRS + Melee Splash + Minion Damage
  • Holy Flame Totem + Combustion
  • Clarity (remover após Eldritch Battery)
  • Shield Charge + Faster Attacks

Core (mid/end)

  • SRS + Unleash + Multistrike + Chance to Poison
  • Desecrate + Offering + Curse setup para boss
  • Determination + Purity of Elements
  • Stone/Chaos Golem + Feeding Frenzy

Prioridade de upgrades

  • Rumi's Concoction
  • The Covenant
  • United in Dream
  • Aul's Uprising (versão budget → ideal)
  • Progenesis (luxo/high budget)

Craft base

  • Boots: chaos res fraturada + avoid ailments
  • Gloves: base com vida/res + essence minion dmg
  • Shield: spell block fraturado + essência de block
  • Sempre manter res capadas (75%+)

Animate Guardian

  • Weapon: Dying Breath
  • Helmet: Leer Cast
  • Body: Belly of the Beast → Doppelgänger Guise
  • Boots: Windscream
  • Gloves: Asenath's Gentle Touch

Spectres úteis

  • Stygian Silverback (frenzy charges)
  • Host Chieftain (power charges)
  • Arena Master (buff de velocidade)
  • Versões “perfect” conforme orçamento/mercado

Plano de progressão

  • Finalizar campanha com labs em dia.
  • Mapas iniciais: estabilidade defensiva + links eficientes.
  • Transição gradual para poison/chaos com The Covenant.
  • Fechar upgrades caros por ordem de impacto/custo.
  • Ajustar setup conforme patch e economia da liga.
