Path of Exile 1 • Witch Necromancer

League Start Mirage — Guia Profissional SRS (Summon Raging Spirit)

Build focada em progressão segura de campanha até mapas, com transição para poison/chaos no endgame. Página organizada para consulta rápida durante leveling e farm.

Visão geral da build

A Witch SRS é um league starter sólido: limpa campanha com conforto, escala bem em mapas e evolui com upgrades graduais. O núcleo inicial usa SRS + Holy Flame Totem; no mid/endgame, migra para setup com poison/chaos e itens-chave.

Classe: Witch Ascendancy: Necromancer Skill principal: Summon Raging Spirit Meta: League Start Mirage

Checklist rápido

  • Cap elemental resistance em 75%+ ao final da campanha.
  • Priorizar atributos em gear: STR/DEX para requirements de gem.
  • Concluir labs (1 a 3) antes de pressionar mapas altos.
  • Usar base de Armadura + Energy Shield para facilitar cores.
  • Manter frascos e links atualizados a cada ato.

Skill Gems por fase

Ato 1–2 (base)

  • SRS + Ruthless (depois trocar para Splash/Minion Damage)
  • Holy Flame Totem + Combustion
  • Mobilidade: Frostblink (ou Flame Dash)
  • Utilitárias: Clarity, Shield Charge, Vitality (opcional)
  • Subir Desecrate em slot secundário no Ato 2

Ato 3–4 (estabilização)

  • Auras: Purity of Elements + Determination
  • Curse: Elemental Weakness
  • SRS + Unleash + Multistrike + Chance to Poison
  • Golems: Stone Golem / Chaos Golem + Feeding Frenzy
  • Ao pegar Eldritch Battery, ajustar mana reservation (remover Clarity)

Mapas iniciais

  • Setup principal de SRS em 4L/5L com foco em dano de minion.
  • Arcanist/utility setup com Desecrate + Offering + Curse para bosses.
  • Guardar Minion Damage para quando fechar 5L/6L apropriado.

Transição poison/chaos

  • Com The Covenant, migrar links para escala de chaos.
  • United in Dream ajuda a fechar chance alta de poison.
  • Ajustar aura/reservation conforme colar e passivas disponíveis.

Equipamentos recomendados (League Start → Endgame)

Prioridade de upgrades

  • Rumi's Concoction (sobrevivência com block).
  • The Covenant (salto forte de dano chaos em minions).
  • United in Dream (sinergia de poison no SRS).
  • Aul's Uprising (versão de aura viável no orçamento inicial e upgrade depois).
  • Progenesis (upgrade de alto custo para suavizar conteúdo difícil).

Craft básico sugerido

  • Botas: base fraturada com chaos res + essence para avoid ailments + movimento.
  • Luva: base fraturada (vida/res) + essence de minion damage.
  • Escudo: base fraturada com spell block + essence para block/avoid ailments.

Animate Guardian e Spectres

Animate Guardian (setup prático)

  • Weapon: Dying Breath
  • Helmet: Leer Cast
  • Body: Belly of the Beast → Doppelgänger Guise
  • Boots: Windscream
  • Gloves: Asenath's Gentle Touch

Spectres úteis

  • Stygian Silverback (frenzy charges)
  • Host Chieftain (power charges)
  • Arena Master (speed buffs)

Progressão por atos (resumo operacional)

Ato 1

Comece com Rolling Magma, migre para SRS cedo, mantenha Holy Flame Totem para suporte de dano no leveling inicial.

Ato 2

Pegar Desecrate, preparar Tempest Shield e matar todos os bandidos conforme plano da build.

Ato 3

Ativar Purity/Determination quando a reserva permitir; buscar soluções de DEX para requisitos.

Ato 4+

Entrar no setup com Unleash e preparar transição gradual para mapas/endgame com foco em resistências e links.
